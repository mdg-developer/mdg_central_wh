Odoo 12 Product Label Report Extension

Module allows to change the sequence, font-size and color of fields to display in barcode label report.


Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

How to use

  • To use this module, we have to provide "Manage Product Variants" rights from "Technical Settings" to login user.
  • Select product(s) from list view or form view.






  • By clicking on "Dynamic Product Label" menu from print button, wizard will popup, and user has to fill appropriate data.
  • In, "Products" tab user will get product list which user has selected from list or form view or can add product(s) in list.


  • In, "Fields" tab user can select fields of product which user need to print in label.


  • In, "Barcode" tab user can select barcode type and display height-width to print barcode in label.


  • By clicking on print button user will get print of label(s) as shown in below figure.
  • We can set currency symbol position `after` in product sale price.


  • Same way, we can set currency symbol position `before` in product sale price.


  • On Click Save button, if update existing design option is not selected, So one pop-up will open where user can give design name and save design.


  • On Click of Product Label Design menu, you can see the list view of saved design.
